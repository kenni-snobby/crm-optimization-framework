# CRM Audit Checklist

## Objective
Ensure the CRM supports accurate reporting, efficient sales execution, and predictable revenue growth.

---

## Data Quality Audit

### Contacts
- All contacts have names
- Email addresses validated
- Duplicate contacts removed
- Contact ownership assigned

### Companies
- Industry field completed
- Company size captured
- Website populated
- Parent-child relationships mapped

### Opportunities
- Deal value populated
- Close date defined
- Opportunity owner assigned
- Stage accurately reflects reality

---

## Pipeline Audit

### Stage Design
- Entry criteria defined
- Exit criteria documented
- No overlapping stages
- Stages align with buyer journey

### Deal Hygiene
- No stale opportunities
- Next steps documented
- Follow-up tasks assigned
- Lost reasons tracked

---

## Reporting Audit

### Core Metrics
- Lead-to-opportunity conversion rate
- Opportunity-to-close conversion rate
- Average sales cycle
- Pipeline coverage ratio
- Forecast accuracy

### Dashboard Review
- Executive dashboard available
- Sales manager dashboard available
- Rep-level dashboard available

---

## Automation Audit

- Lead routing automated
- Task creation automated
- Follow-up reminders active
- Notification workflows configured

---

## Governance Audit

- User permissions reviewed
- Required fields enforced
- Naming conventions documented
- Quarterly CRM review scheduled

---

## Success Criteria

A healthy CRM should provide:

- Accurate forecasts
- Clean reporting
- Consistent pipeline management
- Reliable revenue insights
